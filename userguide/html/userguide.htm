Modem Search

 

The modem search results page is split into three sections.

 

  • Modem information
  • Host Information
  • Flap Information

 

The “text” button on the search results page gives you the same results in an easy to read copy/paste compatible format for use in note keeping / ticket creation.

Results

Above you see the host information. ISG Tools has support for two hosts (CPEs) behind each modem. The host that is at the top usually has most recently pulled an ip address. The second host (if applicable) will be listed on the next line in the table field. Next to both of the MAC addresses you have separate clear buttons which will clear the host from the router and the modem with a single click. In the right hand cell you have the ping test results for both hosts.

Clicking clear will clear the host…

Clear host: 0010.a4fa.9efc

Results

UBR2>clear cable host 0010.a4fa.9efc

uBR Vital Stats & Other uBR CGI’s

 

The ubrcheck script pulls uBR Version Information, CPU Utilization, Uptime, SNR Information (for every blade and port), and connected modem information.

 

Offline, Init, and Maxscreamer and both flaplists work the same way, select the uBR to query the information from and click on submit.

 

 

IP Search & Historical IP Search
(and info on the backend scripts that support them)

 

Using the standard IP search script (ipsearch) you can query the current logs for the ip address you enter. The script parses existing logs from the iplogger script and locates the modem that the ip is behind.. Then it outputs….
 

[ 24.123.123.123 ]   is behind   [ 0020.4089.bb7c ]



Modem information last updated from uBR: Thu Apr 18 21:36:32 EDT 2002

The Flaplists
ISG Tools pulls and clears the flaplist nightly. It will take the results from the list and paste the ubrcheck results for the uBR it was cleared from at the top. There is also a liveflap cgi script that pulls the live flaplist from the uBR of your choice without clearing it.
